MacBook M1 – Landing Page

A modern and interactive landing page inspired by Apple’s MacBook M1.
Built using HTML, CSS, and JavaScript with a strong focus on clean UI,
smooth animations, and user experience.

--------------------------------------------------

FEATURES

• Modern Apple-inspired design  
• Smooth animations and transitions  
• Interactive video section (YouTube embed)  
• Color selection (Silver / Space Gray / Gold)  
• Dynamic price updates  
• Fully responsive layout  
• Smooth scrolling navigation  
• Clean and readable code structure  

--------------------------------------------------

BUILT WITH

• HTML5  
• CSS3  
• JavaScript (Vanilla)

No frameworks or external libraries used.
